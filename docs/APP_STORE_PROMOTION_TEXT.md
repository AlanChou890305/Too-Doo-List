## Usage Notes

**Promotion Text Guidelines:**

- Can be updated anytime without submitting a new version
- Appears at the top of your App Store listing
- Maximum 50 characters (English) or 50 characters (Chinese)
- Should highlight key features and benefits
- Keep it concise and impactful
- Can include emojis but use sparingly
- Should be updated regularly to promote new features or special offers

**Recommended Updates:**

- When launching new major features
- During special promotions or events
- To highlight seasonal features
- To promote user testimonials or achievements
